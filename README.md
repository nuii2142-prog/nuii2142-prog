## 🚀 Featured projects

### 🏷️ [MetaGen Stock Tools](https://github.com/nuii2142-prog/metagen-stock-tools)

AI metadata for stock photos. Upload images and let AI vision (Claude / OpenAI / Gemini) write
the title, description, keywords, and category — then export a ready-to-upload CSV for **14
platforms** (Adobe, Shutterstock, Dreamstime, Freepik, and more). A companion Windows tool
embeds IPTC/XMP/EXIF metadata into copies for Dreamstime, never touching your originals.

🌐 **Live demo (no install):** <https://metagen-stock-tools.vercel.app>

`HTML` · `JavaScript` · `AI Vision` · `ExifTool · IPTC / XMP / EXIF` · `Browser + Windows`

### 🔥 [Nuii Auto Bulk](https://github.com/nuii2142-prog/nuii-firefly-bulk)

A Chrome (Manifest V3) extension that bulk-automates **Adobe Firefly** image generation — queue
prompts from text/CSV, run large batches unattended, and capture full-resolution results (true
**2K** on Firefly Image 5) into a single ZIP. Runs locally, no tracking.

`JavaScript` · `Chrome Extensions (MV3)` · `Service Workers` · `DOM & Network Automation`
